Sakura Sushi Website

Description: The Sakura Sushi Website is a responsive web application developed for a fictional sushi restaurant. The website provides users with a menu of sushi items and allows them to place orders online. It features a clean and user-friendly interface designed to enhance the user experience.

Key Features:

  • Menu Display: The website displays a variety of sushi items with images, descriptions, and prices.
  • Order Placement: Users can add items to their cart and place orders.
  • Customer Information Form: A form is provided for users to enter their first name, last name, and other required information for order processing.
  • Order Summary: A summary of the order is displayed before submission, including the total cost.
  • Confirmation Message: After placing an order, users receive a confirmation message thanking them for their order.
  • Responsive Design: The website is designed to be responsive, ensuring a seamless experience across various devices and screen sizes.

Technologies Used:

  • Frontend: HTML, CSS, JavaScript
  • Backend: Local Storage for data persistence
  • Development Tools: Visual Studio Code, Git
