Library Management System

The Library Management System is a web application developed using Django and Python. It allows librarians to manage books, patrons, and borrowing records efficiently.

Features:

  • Book Management: Allows librarians to add, edit, and delete books from the library database.
  • Patron Management: Provides functionality to manage library patrons, including adding new patrons and updating their information.
  • Borrowing Records: Keeps track of books borrowed by patrons, including due dates and return status.
  • Search Functionality: Allows users to search for books or patrons using various criteria.
  • Responsive Design: Uses Bootstrap for styling, ensuring the application is responsive and accessible on different devices.

Technologies:

  • Python: The core language used for development, providing the backend logic for the application.
  • Django: The web framework used to build the application, providing a robust structure and built-in functionalities.
  • SQLite: The database management system used for storing and managing library data.
  • HTML/CSS/Bootstrap: Used for the frontend development, providing the user interface and styling for the application.

Overall, the Library Management System is a comprehensive web application that demonstrates proficiency in Django, Python, and frontend technologies, providing a functional and user-friendly solution for managing library operations.
