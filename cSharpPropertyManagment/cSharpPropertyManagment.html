Property Rental Management System

Designed to assist property managers in handling rentals, bookings, and tenant management efficiently, this web application offers robust tools to streamline operations in real estate management.

Key Features:

  • Property Listings: Manage and list properties available for rent.
  • Tenant Management: Keep track of tenant details and rental agreements.
  • Booking System: Facilitate the booking of rental properties.
  • Payment Integration: Process payments securely within the platform.
  • Messaging System: Communication between landlords and tenants.
  • Role-Based Access Control: Different permissions for various user roles.
  • Filtering Options: Advanced property and tenant filtering capabilities.
  • Appointment Scheduling: Manage and schedule property viewings and other appointments.

Technologies Used:

  • Frontend: HTML, CSS, JavaScript
  • Backend: C#, ASP.NET MVC
  • Database: SQL Server
  • Docker: Containerization of the application for easy deployment and scaling
