Project Title: Multi-Functional C# Dashboard

Description:

This C# application serves as a versatile dashboard, providing users with quick access to various tools and functionalities. Developed entirely in C#, it showcases my proficiency in the language and my ability to create user-friendly interfaces using Windows Forms.

Key Features:

  • Lotto Number Generators:
    • Generates random lotto numbers for popular games like Lotto Max and Lotto 6/49, including bonus numbers if applicable.
    • Stores the generated numbers and timestamps in a text file for easy access and tracking.
  • CurrencyExchange Calculator:
    • Facilitates conversion between various currencies (CAD, USD, EUR, GBP, TND) using pre-defined exchange rates.
    • Allows users to specify the amount and select the desired currencies for conversion.
    • Displays the converted amount in a clear and user-friendly format.
  • Temperature Converter:
    • Converts temperatures between Celsius and Fahrenheit, offering users the flexibility to choose the input and output units.
    • Provides informative messages based on the converted temperature, such as "Water boils," "Beach weather," or "Extremely Cold Day."
    • Saves conversion history along with timestamps and messages to a text file for reference.
  • IP Address Validator:
    • Verifies the validity of user-entered IP addresses using regular expressions.
    • Provides informative messages, indicating whether the format is valid or invalid.
    • For valid IP addresses, stores them along with timestamps in a binary file for potential future analysis (optional functionality depending on your project goals).
  • Calculator:
    • Provides a basic calculator functionality for simple mathematical operations (addition, subtraction, multiplication, division).
  • File I/O Operations:
    • Utilizes various file I/O techniques (reading, writing, appending) to save and retrieve data from text and binary files, ensuring persistence of generated numbers, conversion history, and IP addresses.

Skills Demonstrated:

  • C# programming language proficiency
  • Windows Forms application development
  • User interface design principles
  • Regular expressions for data validation
  • File I/O operations
  • Object-oriented programming concepts
