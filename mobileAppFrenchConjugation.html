iOS App for French Verb Conjugation

Description: Created an iOS app using Swift that utilizes an API for French verb conjugation. The app provides a user-friendly interface for conjugating French verbs, making language learning more accessible.

Tech:

  • Swift: The core language used for iOS app development.
  • iOS Development: Used for developing the iOS app.
  • API Integration: Integrated an API for French verb conjugation.

Achievements:

  • Developed an intuitive iOS app for learning and practicing French verb conjugation.
  • Integrated a reliable API for accurate and up-to-date verb conjugation information.
  • Received positive user feedback for the app's simplicity and effectiveness in language learning.
