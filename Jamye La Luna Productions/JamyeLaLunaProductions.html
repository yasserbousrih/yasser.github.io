Jamye La Luna Productions - Event Management Platform

Project Overview: A dynamic full-stack web application built for an event management company, featuring a powerful content management system and public-facing website. The platform enables seamless management of entertainers, events, and services through an intuitive admin dashboard while providing an engaging experience for potential clients.

Key Features:

  • Dynamic entertainer profiles with photo galleries
  • Comprehensive service and event catalogs
  • Responsive design for all devices
  • Interactive image galleries with Google Photos integration
  • Real-time content updates
  • Secure authentication system
  • User-friendly content management interface

Technical Stack:

  • Frontend: React.js, React Router, Custom CSS, React Toastify
  • Backend & Infrastructure: Firebase (Authentication, Firestore, Storage), Google Photos API

Core Functionality:

  • Content Management: Create, update, delete profiles and manage events
  • User Experience: Intuitive navigation, fast loading, responsive image handling

Development Highlights:

  • Clean, maintainable code architecture
  • Secure authentication and protected routes
  • Optimized performance and scalable database structure

Business Impact:

  • Streamlined content management
  • Enhanced digital presence and user engagement
  • Efficient administrative workflows and professional service presentation
